The project is organized into **five tasks**, each mapping to one module of the Digital Literacy course. Together they demonstrate practical knowledge of:

- 🎨 Digital tools and visual communication

- 💻 Coding platforms and cloud collaboration
- 📧 Professional email communication and etiquette
- 🔒 Cybercrime awareness and prevention
